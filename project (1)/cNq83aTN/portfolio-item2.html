Tic-tac-toe game

Java, Unbeatable Computer AI

This is a Java based application that allows users to play a tic-tac-toe game against an AI opponent. The AI is designed to be unbeatable when it goes first, meaning it will always achieve at least a draw, if not a win.

The core of the project involves writing Java code to create the game's logic, user interface, and AI algorithm. Understanding object-oriented programming concepts is crucial for maintaining clean code and organizing the different game components. Implementing the rules and logic of tic-tac-toe is fundamental to the project's success. This includes checking for winning conditions, validating user input, updating the game board, and detecting draws.

Representing the game board efficiently in data structures is essential for the AI algorithm to traverse the possible game states. A 2D array was used to represent the game board, updating and outputing on screen after every turn
